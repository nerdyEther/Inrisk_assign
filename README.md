## Setup

1. Clone the repository:
   ```
   git clone https://github.com/nerdyEther/Inrisk_assign.git
   ```

2. Navigate to the project directory:
   ```
   cd Inrisk_assign
   ```

3. Install the required dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```
